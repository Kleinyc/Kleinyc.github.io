与卿共勉书

置顶 | 发表于 2022-04-17 更新于 2022-04-18

壬寅虎年，春分已过，昼长夜短，日渐奔忙。三月将逝，玄鸟又归，四阳渐盛，阴阳薄雷。春耕忙田，良人纸鸢，雨后丰年，平安每天。

与阳同起，红旗相迎，伴月西归，前行不停。四年即末，知有课枯燥无味；身居庠序，见有师别具一格。今夜抒怀，感概良多。各地穿梭来往，见未所见，闻未所闻，自学登堂，慢慢成长。

晨来日见，暮至月回，天高云淡，鹰啼高飞。百年已至，拼搏之时。铭记责任，求真务实，海纳百川，自强不息。肩负使命，不负重托，承先辈“规格严格”之训，做当代“功夫到家”之人。

莘莘学子，恭勤不倦，春夜茫茫，书山跋涉。求学之人，汲知之子，不拒小流，不让细壤。愿承先辈之志，望明大千法理。秋夜已深，笔辍于此，吾意于文，共勉与卿。

如何从大学里脱颖而出

发表于 2022-04-18 更新于 2022-04-19 分类于 大学

“且听风吟”

才搭建好个人Blog，这是第一篇文章，说实话，一直没想好博客搭建好之后先写什么。适逢毕业季了，最近也经常思考这四年来我的得失，加上这学期的先修课也全部结束了，在即将进入研究生阶段的时间里，就先写一下关于我的大学经历以及我对大学的认知吧，这篇文章也会慢慢完善，感觉不是一次就能说清楚的，之后再慢慢写一些技术类的博文了。

整体框架

  • 经历篇

春夜缓行思钟鼓，旅途且停听风吟。——莲荷 D

经历篇

其实《如何从大学里脱颖而出》本身就是由卡尔·纽波特博士写的一本书，我是在大一的时候看了这本书，在此就借鉴一下Doctor的书名。虽然说中国的教育和美国不一样，但是这本书也是非常值得参考的。那么我的大学四年到底经历了什么呢？

在HIT学习

学习应该说是学生最重要的一件事，不要听那些“进了大学就轻松了”的鬼话，这个时间点活出自己是非常不容易的。在哈工大如果没有成绩，其实会错过非常多的事情，其他学校应该也是一样的，虽然说我不是那种学习特别特别出色的人，但是还是本科四年维持在一个可以保研的范围内，结果也是成功保研了。

但是我不是那种拼命学的人，人还是挺跳脱的，所以对力学这种非常考验耐心的学科实在无能为力，学的再认真也就是个80分的水平，所以也及时止损了。大三分方向后选了工程管理的房地产方向，无所谓喜欢与讨厌，但整体还是学的来的，最后是跨专业保研到了管院信管专业，之后就是读两年的研究生了，博士就根据情况而定。

因为很早地认识到“大学、大学、大不了自己学”，上课一直不怎么认真听讲，而且大学的老师讲课非常快，一门课课下花费的时间至少得是上课时间的3-5倍才能学的通透。所以就选了很多奇奇怪怪的课程，包括但不限于“大一选外教上人工智能、不太懂音乐的时候选陶笛演奏、喝自己酿的啤酒这种水课······”，还有各种专业性的课，已小班课为主，课后作业完成的比较认真，大班课太水了，整体拓宽了知识面，所以学习上面也算是坦途吧。这里给出的经验就是，必修课一定考好，多种途径拓宽一下知识面。

竞赛项目

先上图，参加了好多项目比赛，从国一到省级、校级、学院，各种奖都有，这是其中是一部分了，还有一些综合性奖励。面挺广的，有专业性比赛、创新类的、数学建模以及一切奇奇怪怪的比赛，甚至学院组织的寝室辩论赛，我们寝室都拿到了第一名，可以说“哎呦，不错哦”。

但其实比赛是一件既痛苦又快乐的事情，我是非常支持去做比赛的。相比于本科发水论文，比赛的节奏更快一些，我个人认为本科还是挺强调认知广度的，专业性的东西迟早都会学会，混口饭吃的技术并不是很难。当然也只是个人狭隘的见解了，因为大部分需要本科结束去工作，论文就给更显得不是那么重要了，参加一些技术含量不错的比赛挺有含金量的。

最有意思的应该是做项目或者比赛时，一旦解决了问题时的那种成就感。其实包括我在内的很多人都普遍觉得本科的课程好水啊，加上大家也不是特别认真，考个高分，时间一长，最后扔掉了一些知识，也不知道学那些有什么用，毕竟很多毕业偏向于Office的工作并没有非常强的专业性，知道某些知识就可以了，学校里学的深度还是挺深的（认真自学了）。但是做一些建模类的比赛时，就感觉到了，心理变化是从“啊，还能这样搞？”到“之前学到的真的用到了哎”。还有做项目，特别是做算法时，许多的数学概念非常有趣，也在“挑战杯”现场推介会的时候看到了很多优秀的作品，那些同学给我介绍他们的作品时，我也经常会眼前一亮，所以如果有时间愿意折腾，大学完全可以多做一些比赛。

具体就不讲哪些竞赛或者项目了，后续会分享一些关于大创项目、数学建模等相关的技术和组织方式的见解，这些或许会更有用一些。

读书篇

其实关于读书，基于本身的热爱，也是践行着卡尔博士书中的“法则1 不必通读全书，在大学里，你有大量的书要读，要定好目标，并选择关键内容来进行阅读，完成学习任务，这种方法无疑是后天习得的，你必须下点苦功，尽可能的大量阅读，学会找重点，学会快速的浏览资料，提高效率。”

截止到现在，我在学校图书馆里借阅了142本书了，微信读书书架里有64本，去过国家图书馆、天津市图书馆、哈尔滨市图书馆、宝山区图书馆、西安市图书馆等等等等，特别喜欢收藏图书馆的读书卡。自己也有买二十余本书吧，也下载了不少txt和pdf文档到Ipad上，刨除一些重复借阅和借了没看的，应该整体上满足一个书架250-300本的范围了。所以在读书方面还是可以给出一些建议的。

首先我建议，有时间就去泡图书馆吧，这是在18岁之后，很难再有的一段时间了，特别是学校里的环境非常适合读书。

其实我一直都处于一个非常奔忙的状态，感觉自己闲不下来，有时候什么第二天有很多DDL了，我依旧会怀着各种各样的心情去读了好几本书，有时候难以忍受地借了十几本书，花一两个周抛开很多事情，去读书，带着两三支速溶咖啡，在图书馆一坐就是一天。

那么，书里面到底有什么呢？

书里有什么，我还是把它放到修身篇里讲述了。

实践活动

“古往今来多少事，秦汉唐宋土一抔，蝶梦庄周千余恨，国破家亡万斛愁。历史长河多风雨，时代变迁诸事空，何事堪得永传颂？精忠报国在心中。”

或许是从小看了太多的史书，始终有着凝结在心头的爱国主义情怀，我在军训时就加入哈工大国旗护卫队。大一，我是一位队员，每天早上五点半起床，六点在五公寓前集合开始早训，体能、队列、条例、口号……每周五天早训雷打不动，除了一次有事请假外，我全勤出训。听起来其实并没有特别了不起，但是作为一名普通队员，见证了因为早训，很多的战友都退出了这个组织，从一开始的100多人，到大一结束，只有20个人左右还在坚持。18年，国护正式建队，大一学年是十分艰辛，可以说是跌跌撞撞地往前走，在这里，我真正意识到了何为“使命和担当”，为国护旗，为校出旗，不是一句简单的口号，而是我们国护人始终的坚守。同时国护也让我有了很多良好的习惯，影响很深。

在大学里，是可以加入一些组织的，我去加入过不少组织，包括国旗护卫队、合唱团、院学生会、帮扶小组等等。因为大家对学校社团组织的评价褒贬不一，这一方面，我只能说可以加入，但也要慎重，毕竟人的精力是有限的。人毕竟不单单是一个经济人，还有社会属性，一个组织代表着一个圈子，一种文化，好与不好都不能一概而论，毕竟彼之砒霜,吾之蜜糖。在这里我个人建议可以加入像“国旗护卫队、合唱团、兴趣类社团”这样的组织。

然后就是实习，如果不打算读研的话，有时间就早早实习吧。大学还是有很大一部分是通识教育，有些人是要早点毕业，尽快独立的存在，虽说疫情期间出去实习的难度不小，但是办法总比困难多，后续在我也会分享一些我找实习的经历以及注意事项。

修身篇

成为书虫

我读的书里面还是以小说居多，相信我们这个年龄的不会非常多地读一些哲学散文什么的，那些往往是精读的书。小说的节奏非常快，我曾以第三视角去读了《信》，信里有什么呢？我最想理解的是写信是为了寄托什么样的感情，因为我经常给我的朋友们写信写明信片，也收到过不少信，其中真的是我们朋友间最真挚的情感，所以我们也都会一辈子珍藏。东野先生的《信》讲述主人公是一对在父母死后相依为命的兄弟：因工作受伤而失业的哥哥武岛刚志，为了筹备弟弟的学费，铤而走险犯罪杀人；原本成绩优异前途光明的弟弟武岛直贵，因为哥哥突如其来的犯罪，不得不背负杀人犯弟弟的精神债务，坠入社会歧视的轮回。一封封寄自高墙之内的家信，寄托了哥哥对弟弟的无尽牵挂，同时也为弟弟带来了无尽的噩运。弟弟不堪忍受歧视，与哥哥断绝关系······因为东野先生的作品普遍要和犯罪相关，这里的“信”也有一些特殊的意思。

我曾努力以不被影响的态度看了《人间失格》，以“我”的身份感受了《变身》，在错综复杂的社会背景下重新理解了一下所谓的性别问题——《单恋》，同时去探寻了一下《秘密》，到底是什么秘密呢？看了就知道。而《预知梦》又让我想起了自己做的无数个梦，后来养成了一个习惯，如果做梦了，早上醒了的第一件事是在便签上记录下来，说不定就是以后的写作素材。

还有金庸先生的武侠小说，飞雪连天射白鹿 笑书神侠倚碧鸳，武侠世界，恩怨情仇，说不清，道不明。值得讲的还有芥川先生的《竹林中》，对应的电影《罗生门》实在是值得一看，“无可规避的恶”和东野的《恶意》很辉映，反思的过程也是不断提升自己的修养，毕竟“君子慎独”。

我想我未来是，等我老了是非常想去开一个文具店的，因为“除了花以外，枝叶和根，以及被虫咬的痕迹，所有的一切都很美丽。”——《山茶文具店》，小川糸女士的一本治愈系小说，有16封手写的信，主人公也曾是一位叛逆的少女，与阿嬷相伴，又被当作代笔人培养，在青春期的时候叛逆过，最后重新回到故里，又拾起代笔人这一重身份，继续开着山茶文具店，这个过程中，她有了自己的好邻居，好朋友芭芭拉夫人，在写信的过程中还结识了男爵、胖蒂、QP妹妹，甚至最后即将邂逅属于自己的爱情。

村上先生的许多书值得读一读，特别是迷茫的时候，所谓“愚者之虑莫如休憩”，文章中村上式不合理却又那么合理。哪有那么多原因，就像堕入情网时，那是一场犹如以排山倒海之势掠过无边草原的龙卷风一般的迅猛的恋情，就像是绿野上奔跑的熊，还有森林里的狮子，奇奇怪怪的，没有什么原因，因为乌鸦像写字台。从《且听风吟》开始，慢慢品味。

不局限于小说，我也读过《毛选》、《深度工作》、《共产党宣言》以及一些心理学的书籍，还有不少网文，力推《诡秘之主》。喜欢什么就读读什么吧，有时间一定要去读读书，刷短视频、知乎过的太快而营养不足。

那么为什么要多看书呢？哈哈哈哈，哪有什么为什么，读书好不好我不知道，但我知道没有任何不好的事情。书很纯粹，不像知乎、微博以及这篇博文，其实带有很多的主观色彩。你根本不知道有些人写推文的目的是什么，有些看似很不错的分析营销占了不少部分，虚假的信息很多，而我们作为个体，很难基于事实做出理解。而纯文学中的小说体裁讲究纯粹性，“谎言去尽之谓纯”便是所谓的“纯”。也就是说，小说在构思及写作的过程中能去尽政治谎言、道德谎言、商业谎言、维护阶级权贵谎言、愚民谎言等谎言，使呈现出来的小说成品具备纯粹的艺术性。小说的纯粹性是阅读者最重要的审美期待之一。随着时代的发展，不光是小说，整个文学的纯粹性逾来逾成为整个世界对文学审美的一个重要核心。
